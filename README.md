# 🎮 Simon Game — Boss Level Challenge 02

🚀 Live Demo: https://sahranhameed.github.io/The-Simon-Game/

## 📌 About the Project

A browser-based memory pattern game inspired by the classic **Simon** electronic toy, built with **HTML, CSS, JavaScript, and jQuery** as part of the MERN Stack Development training — Boss Level Challenge 02.

---

## 🎮 How to Play

1. Open `index.html` in your browser
2. **Press any key** (or tap/click) to start
3. Watch the color pattern carefully 👀
4. **Repeat the pattern** by clicking the colored buttons
5. Each level adds **one more color** to the sequence
6. Wrong click = **Game Over!**
7. Press any key to **restart**

---

## ✨ Features

- 🌈 4 colored buttons (Green, Red, Yellow, Blue)
- 🔊 Unique sound for each color (Web Audio API)
- 📈 Progressive difficulty — pattern grows each level
- ❌ Game Over animation with red screen flash
- 🎯 Level counter display
- 📱 Works on desktop and mobile (tap to start)

---

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| HTML5 | Game structure & layout |
| CSS3 | Styling, button glow effects, animations |
| JavaScript (ES6) | Game logic, pattern generation |
| jQuery | DOM manipulation, event handling |
| Web Audio API | Sound generation for each button |

## 🚀 Getting Started

```bash

open index.html
```

---

## 🎯 Game Logic Flow

```
Start → Generate Random Color → Flash + Sound
      → User Clicks → Check Pattern
      → Correct? → Next Level → Add Color
      → Wrong?   → Game Over → Restart
```
